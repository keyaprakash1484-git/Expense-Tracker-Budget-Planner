# Expense Tracker & Budget Planner (C++)

## Overview
This is a simple command-line Expense Tracker and Budget Planner built using C++. It helps users record daily expenses, monitor their monthly budget, and view how much money has been spent and saved.

## Features
- User login with username
- Set a monthly budget
- Add expenses with date, category, and amount
- View all recorded expenses
- Check total budget, total spending, and remaining savings
- View category-wise expense summary
- Save expense data to a file
- Load previously saved data when the program starts

## Technologies Used
- C++
- File Handling
- Structures (`struct`)
- Vectors
- Loops and Conditional Statements

## How to Run
1. Compile the program using any C++ compiler.
2. Run the executable.
3. Enter your username and monthly budget.
4. Use the menu to add expenses, view records, and check your budget status.
5. Choose **Save & Exit** to store your data.
